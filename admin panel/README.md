# BulkSend Admin Panel

React + Vite admin dashboard for the BulkSend Firebase project.

## Commands

```bash
npm install
npm run dev
npm run lint
npm run build
npm run deploy
```

## Firebase

The app uses the `mailtracker-demo` Firebase project and reads users from the `email_data` collection. Google sign-in is enabled for:

- `chatspromo-admin.pages.dev`
- `mailtracker-demo.firebaseapp.com`
- `mailtracker-demo.web.app`
